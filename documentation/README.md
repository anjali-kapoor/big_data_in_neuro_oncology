# Documentation

These files are **for human readers** (interpretability, methods text, and navigation). They are **not** required to execute the pipeline: the executable source of truth for extraction fields is **`main_workflow/schemas.py`**, and for screening criteria plus the dataset allowlist is **`Prompts/dataset_screening_instructions.txt`**.

- **`COMPLETE_WORKFLOW.md`** — End-to-end steps (preprocessing through extraction), including Batch API commands.
- **`SCHEMA_REFERENCE.md`** — Structured extraction fields overview.
- **`FIELD_REFERENCE.md`** — Field definitions and rationale.
- **`SUPPLEMENTAL_SCHEMA_TABLES.md`** — Tabular schema summary aligned with the supplement.
